# Geometry

Библиотека классов для работы с геометрическими фигурами на плоскости (C++17).  
Реализована иерархия классов: `Point`, `Line`, `Shape`, `Polygon`, `Ellipse`, `Circle`, `Rectangle`, `Square`, `Triangle`.  
Поддерживаются операции: вычисление периметра/площади, проверка принадлежности точки, сравнение фигур (равенство, подобие, конгруэнтность), геометрические преобразования (поворот, отражение, гомотетия).

## Возможности

- **Точка** (`Point`) — сравнение, длина отрезка, середина, поворот.
- **Прямая** (`Line`) — создание по двум точкам, по угловому коэффициенту, сравнение, пересечение, отражение точки.
- **Многоугольник** (`Polygon`) — произвольное число вершин, проверка выпуклости, площадь, периметр, принадлежность точки.
- **Эллипс** (`Ellipse`) — фокусы, эксцентриситет, директрисы.
- **Круг** (`Circle`) — радиус, центр.
- **Прямоугольник** (`Rectangle`) — центр, диагонали.
- **Квадрат** (`Square`) — вписанная/описанная окружность.
- **Треугольник** (`Triangle`) — центр масс, ортоцентр, прямая Эйлера, окружность девяти точек.

Все фигуры поддерживают виртуальные методы:
- `perimeter()`, `area()`
- `operator==`, `isCongruentTo`, `isSimilarTo`
- `containsPoint()`
- `rotate()`, `reflect()`, `scale()`
